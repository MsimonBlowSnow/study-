1.分支的使用:
	当你想提交代码进行备份，又不想让别人看到，就可以使用分支。
	当你完成你的代码，可以打破分支，让别人访问到。

默认在主分支，可以使用下面命令去创建分支
git branch 分支名	//创建一个分支
git branch        //查看所属分支,你在的分支前面有一个星号。
git checkout 分支名	//切换分支
git merge 分支名	//合并分支
git branch -d 分支名	//删除分支(注意：一定要在其他的分支上才能删除改分支。
)

合并分支起来冲突(解决方案)
在冲突文件里面会有如下信息:
	两次都有代码
	============head
	dafadfjadf 	//这是主分支的代码
	+++++++++++++
	adfadffaddfdf 	//这是ming分支的代码
	》》》》》》》ming
把你需要的代码留下，并在做一次提交即可。
